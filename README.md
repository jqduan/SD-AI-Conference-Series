# 随机动力系统与人工智能会议系列回顾
# Retrospective: Stochastic Dynamics and Artificial Intelligence Conference Series

这是一个记录“随机动力系统与人工智能/机器学习”系列研讨会的档案库。该系列会议旨在探讨随机动力系统、随机分析以及数据科学热点问题中的新研究。
This repository serves as an archive for the "Stochastic Dynamics and Artificial Intelligence/Machine Learning" workshop series. It aims to explore new research in stochastic dynamical systems, stochastic analysis, and hot topics in data science.
