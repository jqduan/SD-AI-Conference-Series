## 会议概览 / Overview
[cite_start]本次研讨会由大湾区大学段金桥教授等人组织。会议安排了19场学术报告，围绕随机动力系统理论与算法、机器学习理论与算法研究中的最新进展进行了深入交流，涉及贝叶斯神经网络算法、保结构随机动力系统算法等内容 [cite: 21]。
Organized by Prof. Duan Jinqiao and others, this workshop featured 19 academic reports. Experts discussed the latest progress in stochastic dynamical systems theory/algorithms and machine learning, covering topics like Bayesian neural network algorithms and structure-preserving stochastic dynamical system algorithms.
